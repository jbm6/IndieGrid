Project Overview

IndieGrid, a program for small movie theather reservations.

This is a Java-based console application developed for my Final Project in ITCS 3112. The system manages a movie theater seating layout using a 5x6 2D array (30 seats total). It allows users to view showtimes, check seat availability, and make reservations.

Key Features
Seating Management: Uses a coordinate system (Row 1-5, Column 1-6) to manage 30 seats.

Real-time Updates: Visualizes seats using 'O' for available and 'X' for reserved.

Admin Mode: A password-protected area that allows staff to reset the entire seating chart for a new movie.

OOP Principles: Utilizes Encapsulation (private data members) and Inheritance (Admin extends User).


Project Instructions

How to Run:

Open the project folder in your IDE (like IntelliJ, Eclipse, or VS Code) and run the IndieGrid.java file.

Alternatively, compile all files and run the IndieGrid class.

How to Operate:

Menu: Type the number (1-4) and hit Enter to navigate.

Booking: Select "2" to reserve a seat, then enter the row (1-5) and column (1-6).

Admin Mode: Select "3" and enter PIN 1234 to reset the seating chart.
